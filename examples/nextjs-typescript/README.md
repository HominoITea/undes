# Next.js + TypeScript Example Templates

These files are detailed example templates for a production-like React + Next.js + TypeScript project.

## Included files

- `.ai.env.example`
- `ai/context.json.example`
- `ai/llms.md.example`
- `.cursorrules.example`
- `CLAUDE.md.example`

## How to use

From your project root:

```bash
cp examples/nextjs-typescript/.ai.env.example .ai.env
cp examples/nextjs-typescript/ai/context.json.example ai/context.json
cp examples/nextjs-typescript/ai/llms.md.example ai/llms.md
cp examples/nextjs-typescript/.cursorrules.example .cursorrules
cp examples/nextjs-typescript/CLAUDE.md.example CLAUDE.md
```

Then adapt modules, routes, and domain-specific details to your real codebase.
